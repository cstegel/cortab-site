# cortab-web

Website for converting chords to tabs and vice-versa.

## Getting Started

In seperate terminals, run each of the following:

- `cargo make serve`
    - Starts the http server.
- `cargo make watch`
    - Rebuilds on changes.

Then navigate to `localhost:8080` in your browser.
